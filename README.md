## Overview
The Pokedex API allows developers to fetch structured information about Pokémon from all generations. It is ideal for apps, games, or educational tools that need Pokémon data.

## Features
- Retrieve Pokémon details by ID or name
- Get Pokémon types, abilities, base stats
- Support for regional forms and variants
- Search Pokémon by type or ability
- Lightweight JSON responses
